## serving locally

```
docker run --rm -v=$(pwd):/srv/jekyll -p 4000:4000 -it jekyll/jekyll /bin/bash
jekyll serve --force_polling
```

viewable on [http://localhost:4000](http://localhost:4000)

admin on [http://localhost:4000](http://localhost:4000)
